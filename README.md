**command interpreter**

- Create a new object
    EX: create class names  attributes

- Update an object
    EX: update class name id  attributes

- Retrieve an object
    EX: show class name id 

- Count an object items
    EX: class name.count()

- Destroy an object
    EX: destroy class name id
